# 金融管理全栈后台

## 运行环境：nodejs、mongoDB
### 应用到的主要技术：JsonWebToken、passport(解释token)、Vue-cli、axios拦截器、跨域、路由守卫、vuex、element-ui框架、

## 一次性运行整个项目，包括后台数据
- 本项目根目录，运行`npm run dev`

## nodejs运行服务器（提供接口）
- `npm i` 
- `npm run server` 

## 运行vue项目
- 进入到client文件夹，运行`npm i`
- `npm run dev` 开启项目
